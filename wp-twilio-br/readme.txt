== Description ==
This plugin lets you to add the twilio client browser recording functionality to your WordPress site. 
== Installation ==
1. Download the plugin
2. Upload the wp-browser-record folder to your site's wp-content/plugins directory
3. Activate the plugin in the WordPress Admin Panel
4. Add your twilio account details
